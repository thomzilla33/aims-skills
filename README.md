# AIMS-OS Claude Skills

Skills for the AIMS-OS team. Install once, available in all your Claude Code sessions.

## Skills included

| Skill | Purpose |
|---|---|
| `aims-os-feature-versioning` | Versioning system — scope ladder, V1/V1.5/V2 tagging, version-map.md |
| `aims-os-scope-toggle` | The V1 / Full Vision toggle component (HTML + React) |
| `aims-os-eng-handoff-spec` | Engineering handoff spec with acceptance criteria per tier |
| `aims-os-ds-component` | Design system component workflow — Figma extraction → implementation |
| `aims-os-prototype-screen` | HTML prototype screen conventions (ScreenLayout, ListViewSection) |
| `aims-os-versioning-quality` | Quality gate — runs before any versioning/handoff output is delivered |

## Install

```bash
git clone https://github.com/thomzilla33/aims-skills.git
cd aims-skills
./install.sh
```

## Update

```bash
cd aims-skills
git pull
./install.sh
```
